# Portfolio Website 

## Overview 
A simple, responsive portfolio website built with Vanilla JavaScript, HTML and CSS. This project demonstrates core web development concepts including:

- Multiple heading sizes (`<h1>` through `<h4>`)
- Images with margin and padding
- Interactive links
- Multiple `<div>` containers
- Custom background colors
- Responsive design for all screen sizes

-------------------------------------------

## Features

| Feature | Description |
|---------|-------------|
| **Responsive Layout** | Works on desktop, tablet, and mobile |
| **Clean Design** | Minimalist aesthetic with pleasant color scheme |
| **Interactive Links** | Hover effects and smooth transitions |
| **Styled Images** | Images with custom margins, padding, and borders |
| **Semantic HTML** | Proper heading hierarchy and accessible structure |

## Common Issues & Solutions

| Issue  |	Solution |
|--------|-----------|
| Image not showing | Check the image URL is correct and accessible |
| Layout breaks on mobile |	Ensure viewport meta tag is in <head> |
| Colors look different | Browser default styles - add CSS reset |
| Links don't work | Add actual URLs to href attributes |

## Key Learning Takeaways

It's important to understand:

+ HTML Structure - Organizing content with headings, paragraphs, and divs

+ CSS Styling - Using design tokens, margin, padding, borders, and background colors

+ Responsive Design - Making layouts work on all screen sizes

+ Image Handling - Properly sizing and styling images

+ Interactive Elements - Creating links and hover effects
